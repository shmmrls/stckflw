## Overview

StockFlow is a comprehensive grocery management system designed for both individual consumers and grocery store administrators. The platform helps track inventory, monitor consumption patterns, manage expiry dates, and optimize grocery operations through data-driven insights.

### Key Benefits
- **Real-time inventory tracking** with automated expiry alerts
- **Consumption monitoring** and usage analytics
- **Gamified experience** with points and rewards system
- **Group collaboration** for shared households
- **Supplier management** and purchase order tracking
- **Waste reduction** through intelligent inventory management

## Consumer/User Guide

### Registration & Login

#### Creating an Account
1. Click "Sign Up as Consumer" on the homepage
2. Fill in the registration form with:
   - Full name
   - Email address
   - Password (8-12 characters, must include uppercase, lowercase, number, and special character)
   - Contact information
3. Verify your email address
4. Complete your profile setup

#### Login Process
1. Click "Consumer Login" on the homepage
2. Enter your email and password
3. You'll be redirected to your personal dashboard

### Dashboard Overview

The consumer dashboard provides a comprehensive view of your grocery management:

#### Key Metrics
- **Total Items**: Number of items in your inventory
- **Total Quantity**: Sum of all item quantities
- **Near Expiry**: Items approaching expiry date
- **Expired Items**: Items that have expired

#### Quick Actions
- **Add Item**: Quickly add new items to inventory
- **View Items**: Browse and manage your inventory
- **My Groups**: Manage household or group inventories
- **Meal Ideas**: Get recipe suggestions based on available items
- **Shopping List**: Create and manage shopping lists
- **Categories**: Organize items by categories
- **Reports**: View consumption and waste analytics
- **Rewards**: Track points and earned badges
- **Activity**: View recent inventory changes
- **Waste Tracking**: Monitor spoiled and expired items
- **Notifications**: Configure alert settings

### Core Features

#### Item Management
**Adding Items**
1. Click "Add Item" from dashboard or navigation
2. Fill in item details:
   - Item name
   - Category
   - Quantity and unit
   - Expiry date
   - Purchase price (optional)
   - Group assignment (for shared households)
3. Save the item to your inventory

**Managing Items**
- **View Items**: Browse all items with search and filter options
- **Edit Items**: Update item details, quantities, or expiry dates
- **Consume Items**: Log when items are used/consumed
- **Delete Items**: Remove items from inventory

**Barcode Scanning**
- Use the built-in barcode scanner to quickly add items
- Supports common barcode formats
- Auto-fills item information when recognized

#### Group Management
**Creating Groups**
1. Navigate to "My Groups"
2. Click "Create New Group"
3. Set group name and type (household, office, etc.)
4. Invite members via email

**Group Features**
- Shared inventory management
- Collaborative consumption tracking
- Group activity logs
- Combined waste analytics

#### Consumption Tracking
**Logging Consumption**
1. Select an item from your inventory
2. Click "Consume"
3. Enter quantity consumed
4. Add notes (optional)
5. Save to update inventory

**Consumption Analytics**
- Track usage patterns over time
- Identify frequently consumed items
- Monitor consumption by category

#### Waste Management
**Tracking Waste**
- Log spoiled items
- Record expired items
- Track waste reasons
- Generate waste reduction reports

**Waste Analytics**
- Waste percentage calculations
- Trend analysis
- Cost impact assessment

**Badges**

#### Shopping Lists
**Creating Lists**

2. Add items manually or from inventory suggestions
3. Set quantities and priority levels
4. Share lists with group members

**List Features**
- Auto-suggestions based on consumption patterns

#### Meal Suggestions

#### Reports & Analytics

**Export Options**

## Troubleshooting

### Common Issues

#### Login Problems
**Issue**: Cannot login to account
**Solution**:
1. Verify email and password are correct
2. Check if account is activated
3. Reset password if needed
4. Clear browser cache and cookies
5. Try different browser

#### Registration Issues
**Issue**: Account registration fails
**Solution**:
1. Check email format is valid
2. Ensure password meets requirements (8-12 chars, uppercase, lowercase, number, special character)
3. Verify all required fields are completed
4. Check for duplicate email address

#### Inventory Not Updating
**Issue**: Changes not reflected in inventory
**Solution**:
1. Refresh the page
2. Check internet connection
3. Verify you have proper permissions
4. Clear browser cache
5. Contact support if issue persists

#### Barcode Scanner Not Working
**Issue**: Barcode scanning fails
**Solution**:
1. Ensure camera permissions are granted
2. Check lighting conditions
3. Clean barcode surface
4. Try manual entry as alternative
5. Update browser if needed

#### Notifications Not Received
**Issue**: Not receiving email alerts
**Solution**:
1. Check spam/junk folder
2. Verify email address in settings
3. Ensure notification preferences are enabled
4. Check email server status
5. Add StockFlow to email whitelist

### Performance Issues

#### Slow Loading Times
**Solutions**:
- Check internet connection speed
- Close unnecessary browser tabs
- Clear browser cache
- Try during off-peak hours
- Update browser to latest version

#### Mobile Responsiveness
**Tips**:
- Use latest mobile browser version
- Ensure stable internet connection
- Rotate device for better viewing
- Use mobile app if available

### Data Issues

#### Lost Data
**Prevention**:
- Regularly save work
- Use stable internet connection
- Don't close browser during operations
- Export important data regularly

#### Sync Issues
**Solutions**:
- Refresh the page
- Check internet connection
- Log out and log back in
- Clear browser cache
